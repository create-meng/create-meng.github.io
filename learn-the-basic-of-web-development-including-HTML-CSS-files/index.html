HTML

一、HTML 基础

定义：超文本标记语言，用于创建网页。

作用：构建网页结构和内容。

1. HTML 文档结构

  • <!DOCTYPE html>：声明 HTML5 文档类型。
  • <html>：根元素。
  • <head>：包含元数据。
  • <body>：包含可见内容。

2. 基本标签

  • 标题标签：<h1> - <h6>
  • 段落标签：<p>
  • 链接标签：<a> 示例链接
  • 图片标签：<img>
  • 列表标签：<ul>、<ol>
  • 表格标签：
    表头 1 表头 2
    数据 1 数据 2
  • 表单标签：

二、HTML 语义化标签

  • <header>：页面头部
  • <nav>：导航区域
  • <main>：主要内容
  • <article>：独立内容块
  • <section>：内容区域
  • <aside>：侧边栏
  • <footer>：页面底部

CSS

一、CSS 基础

定义：层叠样式表，控制 HTML 外观。

作用：实现网页样式设计。

1. CSS 引入方式

  • 内联样式
  • 内部样式表
  • 外部样式表

2. CSS 选择器

  • 标签选择器：p { color: red; }
  • 类选择器：.my - class { background: yellow; }
  • ID 选择器：#my - id { font - size: 20px; }

二、CSS 常见属性

1. 文本属性

颜色、字体大小、对齐等。

2. 背景属性

背景颜色、图片等。

3. 边框属性

边框样式、宽度等。

4. 盒模型属性

宽度、高度、内边距、外边距。
